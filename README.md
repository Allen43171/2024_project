## 程式
 - 主程式
   - main_and_function/main_script.py
   - 主程式會呼叫其他程式
 - 爬蟲程式碼
   - main_and_function/web_crawler.py
 - 文字雲程式

 - 資料處理程式
   - main_and_function/data_preprocess.py

   - temporary_data_processor.py
 - 繪圖

 - 提取常用字

## 使用方式
1. 下載所有python程式
2. 執行main_script.py
3. cmd介面輸入要爬蟲的關鍵字即可
4. 爬蟲期間，勿移動滑鼠
